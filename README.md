# dmtk - da minimalist toolkit

A dumb toolkit.
It should work on any X11 system.

Some features:

- UTF-8 support
- Draw images from a buffer, or from a PNG/JPG/QOI/BMP/GIF file



## Compile and install

dmtk depends only on Xlib.

```
$ make
# make install
```
## Use

See mtkclock and mtkin folders for examples.
